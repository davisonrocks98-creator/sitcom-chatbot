🎭 Sitcom AI Chatbot

An interactive AI-powered chatbot that lets you chat with your favorite sitcom characters in a WhatsApp-style interface.
Built using Streamlit and powered by Groq LLM API, this app simulates realistic conversations with multiple characters.
✨ Features
💬 One-to-One Chat Mode – Talk to a single character
👥 Group Chat Mode – Multiple characters reply in a conversation
🎭 Multiple Characters – Includes iconic sitcom personalities
🖼️ Profile Pictures (DPs) for each character
📱 WhatsApp-style UI
⚡ Fast AI Responses using Groq API
🔐 Secure API Handling using Streamlit Secrets
🧠 Tech Stack
Python
Streamlit
Groq API (LLM)
HTML/CSS (for UI styling)

🚀 Deployment

Deployed using Streamlit Community Cloud:

Connect your GitHub repo
Add secrets in deployment settings
Deploy instantly
